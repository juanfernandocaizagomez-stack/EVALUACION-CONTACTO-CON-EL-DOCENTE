# EVALUACION-CONTACTO-CON-EL-DOCENTE
tarea del autonomo 3

Nombre del proyecto:
Generador de Contraseñas Seguras

Integrantes:
Mi persona (Juan Fernando Caiza Gomez)

Objetivo del sistema:
Crear una herramienta sencilla y práctica que ayude a los usuarios a generar contraseñas seguras de manera rápida, contribuyendo a proteger su información personal y sus cuentas en línea.

Funcionalidades:
El sistema permite generar contraseñas aleatorias y seguras según las preferencias del usuario. Se puede elegir la longitud de la contraseña y decidir si incluir letras mayúsculas, minúsculas, números y caracteres especiales. Además, cuenta con una opción para copiar la contraseña generada de forma rápida y facilita la creación de claves más difíciles de adivinar, mejorando así la seguridad digital del usuario.
